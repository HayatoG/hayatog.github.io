# hayatog.com.br

Portfólio pessoal de Guilherme "hayatog" Oliveira de Souza.
Construído com [Astro](https://astro.build) — estética **KatanaMode** (nanquim sobre creme).

## Desenvolvimento

```bash
pnpm install
pnpm dev      # http://localhost:4321
pnpm build    # gera dist/
```

Deploy via Vercel (build automático no push).
